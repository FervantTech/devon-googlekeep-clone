# Google Keep Clone

A simplified clone of Google Keep built using HTML, CSS, and JavaScript. This project was created to demonstrate front-end web development concepts including DOM manipulation, event handling, local storage, and responsive user interface design.

## Features

- Create new notes
- Edit existing notes
- Archive notes
- Display notes in a clean card layout
- Modal for editing notes
- Tooltips for action buttons
- Notes are saved using Local Storage
- Sidebar with hover animation
- Responsive layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
